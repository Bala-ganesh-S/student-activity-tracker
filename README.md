# Student Activity Tracker

A simple web-based application to manage and display student activities and events.

## 🚀 Live Demo
https://bala-ganesh-s.github.io/student-activity-tracker/

## 📌 Features
- Add and manage student activities
- Display events in a clean interface
- Lightweight and easy to use
- Fully browser-based (no backend)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html → Main application file

## 🎯 Purpose
This project was built to practice frontend development and understand how event-based systems work in web applications.
